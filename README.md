# LLM Recipe Builder (CLI)

A command-line tool that generates cuisine-specific recipes using available
ingredients, powered by Google's Gemini LLM.

## Features
- Reduces food waste by prioritizing available ingredients
- Strict cuisine adherence
- Structured JSON output
- Clean CLI display

## Tech Stack
- Python
- Google Gemini API
- Prompt Engineering

## Setup
```bash
git clone https://github.com/yourname/llm-recipe-builder
cd llm-recipe-builder
